# 订单

# 项目结构

```
app/
├── src/main/java/com/example/orderapp/
│   ├── OrderActivity.kt                    # 主Activity
│   ├── ui/
│   │   ├── left/
│   │   │   ├── OrderTypeFragment.kt       # 左侧Fragment
│   │   │   └── OrderTypeAdapter.kt        # 左侧适配器
│   │   ├── right/
│   │   │   ├── OrderListFragment.kt       # 右侧Fragment
│   │   │   ├── adapter/
│   │   │   │   ├── OrderAdapter.kt        # 订单列表适配器
│   │   │   │   └── StatusTabAdapter.kt    # 状态标签适配器
│   │   │   └── view/
│   │   │       ├── OrderItemView.kt       # 订单项自定义View
│   │   │       └── StatusTabView.kt       # 状态标签View
│   │   └── detail/
│   │       └── OrderDetailActivity.kt     # 订单详情页
│   ├── viewmodel/
│   │   ├── OrderViewModel.kt              # ViewModel
│   │   ├── OrderRepository.kt             # 数据仓库
│   │   └── OrderDataSource.kt             # 数据源
│   ├── model/
│   │   ├── Order.kt                       # 订单数据类
│   │   ├── OrderType.kt                   # 订单类型
│   │   ├── OrderStatus.kt                 # 订单状态
│   │   └── Response.kt                    # 响应数据
│   ├── network/
│   │   ├── ApiService.kt                  # 网络接口
│   │   ├── RetrofitClient.kt              # Retrofit客户端
│   │   └── ApiResult.kt                   # API结果封装
│   └── utils/
│       ├── Extensions.kt                  # 扩展函数
│       └── Constants.kt                   # 常量
└── res/
    ├── layout/                            # XML布局文件
    │   ├── activity_order.xml
    │   ├── fragment_order_type.xml
    │   ├── fragment_order_list.xml
    │   ├── item_order_type.xml
    │   ├── item_order.xml
    │   ├── item_status_tab.xml
    │   └── view_order_item.xml
    └── values/                            # 资源文件
```
